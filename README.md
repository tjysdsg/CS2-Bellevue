# Heightmap
1. Open https://terraining.ateliernonta.com/ and import [terraining-settings.json](terraining-settings.json) config
2. Download heightmap

# Satellite Image

Resolution of satellite image from the website above is too low. Let's use google earth engine to get more high quality image for free.

1. Go to https://code.earthengine.google.com/ and run code in [google-earth-engine.js](google-earth-engine.js)
2. In tasks tab, click run on the "Satellite Image" task
